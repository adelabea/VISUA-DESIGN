The gif is located on the Basket page, and It has a promotional function.

I was designed using a background image and superimposing the different phases of the brand slogan, which are: "I'm brave, I'm sociable, I'm ready. I'm In".

PROCESS

  1. 1. After creating a canvas in Photoshop with the desired size (1024x415px), I placed a photo on it following the size of the canvas.
  2. 2. I created a purple layer in Linear Burn blending more.
  3. 3. Using the "Motion" workspace in Photoshop, I created the timeline with the slogan sentences divided into four different layers.
  4. 4. I exported for Web in GIF format (Looping option: Forever).
