All the images/photos used in this project are being edited in Photoshop and Illustrator.

Also, all of them are in RGB colour mode, and they have been resized to the actual needed size in a 72ppi resolution. In the last step, I proceeded with "exported for Website".

By doing all this process, I could optimize all the images to a lighter weight and improve the loading of the web page.

To download all the editable PSD files, please click here.

IMAGE 01 - LOGO

The logo was created in Illustrator to have it vectorial. It is designed from the corporate typography, editing both "I" to create two people speaking. The speech bubble is a thunder symbol because it represents the "conversation" and "energy" insights.

Afterwards, the logo was adapted within a proportional grid to finalize the aesthetic. To be placed on the website, the logo was exported in an SVG file, due to SVG format is much more powerful than other image formats we can use on the web, as we can manipulate them with code. Also, as it is a vector file, it will never get pixilated.

IMAGE 05 - Collage composition

Download IMAGE 05 JPG optimized file

  1. 1. I created a canvas in Photoshop (1040x500px).
  2. 2. I made a colour background with the official palette.
  3. 3. I cropped the camera photo to remove its background. After that, I resized it and placed it into the file.
  4. 4. I distorted the image and applied a vector mask to edit the colour properties - changing the red tone to purple.
  5. 5. I placed the topography in different layers: on the top of all the images (outlined font) and on the bottom (solid colour font). This way I created an effect of the type changes when it touches an image.

IMAGE 06 - Banner

Download IMAGE 06 JPG optimized file

  1. 1. I selected three photos to promote three different events.
  2. 2. After creating a canvas with the desired size, I placed these photos and resized them to fit in the canvas.
  3. 3. I create a polygonal shape that occupied 1/3 of the width (blue shape in the screenshot on the top), and I multiplied it by three.
  4. 4. I introduced the three images inside their polygonal shape to get the shape.
  5. 5. I exported for Web.
